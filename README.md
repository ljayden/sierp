프리하게
ERP,그룹웨어,인력 관리 시스템의 탈을 쓴 프리랜서,헤드헌팅 업체의 커뮤니티! 
클라이언트트와 개발자를 연결해주는 플렛폼이 아닌 헤드헌팅 업체와 프리의 커뮤니티. 모든 헤드헌팅 업체와 프리개발자들이 가입하게 만들자.
그룹웨어 기능은 부가적인 것이다! 그룹웨어 시스템으로 가지 말자! 


프리하게 3단계 개발 로드맵       
     
1. 혼자서 : 헤드헌팅 업체의 프로젝트 & 인력 관리 툴 
      - 업무효율 향상, 이력관리, 업무 공류  
     
2. 같이 : 
      - 파트너사와 선택적 동맹(협약?), 헤드헌터들과 함께 고객사와 개발자 최신 정보, 평가 공유 기능
  
  
3. 더 넓게 (프리하게 사이트 오픈) : 일반 유저들에게 프리 구직 정보 노출 , 구직 커뮤니티, 업체평가, 개발자 평가 건강한 생태계 조성  
            

- 외부로부터가 내부로부터.. 업무 정리가 곧 업무? 업체에겐 능률을... 개발자에겐 정보를..

- 1,2단계에서의 수익 모델 : 월회비 + 계정비 + 파트너사 지정비
  3단계에서의 수익 모델 : + 구인 광고 비용, 사이트 트래픽을 이용한 ...
  
  
  
3단계로 가기 위한 마케팅은?
  헤드헌팅업체들이 마니 마니 써주면 자동으로 홍보가 된다. 채용 추천 메일 발송시 페이지 접근
   클린 후기 - 계약됐던 이력이 있는 개발자만(이메일로 매핑) 프로젝트 or 고객사 or 헤드업체 후기를 쓸 수 있다.
